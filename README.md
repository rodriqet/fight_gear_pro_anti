# 🥋 FightGear Pro Anti

**Demo en producción:**  
👉 https://fightgearproanti.vercel.app/

---

## 📌 Descripción

**FightGear Pro Anti** es una aplicación web de eCommerce enfocada en la venta de equipamiento para artes marciales (BJJ, MMA, Boxeo, Muay Thai, etc.).

Este proyecto tiene un **enfoque puramente experimental y formativo**. Su objetivo principal es estudiar y comprender:

- El flujo de creación de estructura de una aplicación mediante IA
- La generación asistida de arquitectura y componentes
- El proceso de despliegue en infraestructura externa
- La automatización de builds y entornos de producción

No se trata de un producto comercial real, sino de un laboratorio técnico.

---

## 🎯 Objetivo del proyecto

El propósito de este proyecto es analizar:

1. Cómo una IA puede generar la estructura base de una aplicación eCommerce.
2. Cómo se organizan:
   - Componentes
   - Rutas
   - Lógica de productos
   - Carrito de compra
   - UI/UX básica
3. Cómo se realiza el despliegue en un entorno cloud externo.
4. Cómo funciona el pipeline de integración con repositorios Git.

Este proyecto nace por **mera curiosidad técnica y aprendizaje práctico**.

---

## 🧠 Enfoque técnico

El proyecto se centra en:

- Arquitectura generada asistida por IA
- Organización modular del código
- Simulación de flujo de compra
- Despliegue continuo
- Gestión de dominio público

No incluye procesamiento real de pagos ni backend productivo.
